# vue-antv-g6-demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

参考链接：http://antv.alipay.com/zh-cn/g6/2.x/demo/flow/work-flow.html

下载插件
```
npm install @antv/g6@2.2.0
```
